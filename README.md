# Shiksha ERP — School Management SaaS Platform

Premium multi-tenant School ERP SaaS built for Bangladesh, scalable globally.

## Tech Stack
- Next.js 15 (App Router) + TypeScript + Tailwind CSS
- Supabase (PostgreSQL + Auth + Storage + RLS)
- Zustand, TanStack Table, Recharts, React Hook Form + Zod

## Quick Start
1. `cp .env.local.example .env.local` and fill credentials
2. Run `supabase/schema.sql` then `supabase/seed.sql` in Supabase SQL editor
3. `npm install && npm run dev`
